# @repo/shared

Shared types, constants, and utilities across frontend and backend.

## Contents

- **types.ts** - API request/response types, data models
- **constants.ts** - App-wide constants
- **utils.ts** - Common utility functions

## Usage

```typescript
import { 
  IssueCredentialRequest,
  DEGREE_TYPES,
  formatDate,
  shortenHash
} from '@repo/shared';
```
